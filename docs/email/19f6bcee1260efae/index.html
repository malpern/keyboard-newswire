OpenAI releases Codex Micro physical keypad

OpenAI ships a physical keypad for Codex agents: OpenAI revealed Codex Micro, a customizable mini keypad designed to make it easier to command agents. The keypad lets you prompt Codex verbally with a button, launch common workflows with a joystick, and adjust the AI’s reasoning power with a knob. It’s the first piece of hardware from OpenAI, but won’t be the last. Watch the viral release video
